# Java

## Prerequisites

None.

## Course Materials:

1. Introduction - [Oracle Documentation](http://docs.oracle.com/javase/tutorial/java/index.html)
2. Java Basics Courses:
    - [Princeton Intoduction to Java Programming](http://introcs.cs.princeton.edu/java/home/)
    - [Codeacademy](https://www.codecademy.com/courses/learn-java)
3. Books:
    - [Head First Java](http://www.headfirstlabs.com/books/hfjava/) OR
    - [Thinking in Java](http://www.amazon.in/Thinking-Java-4th-Eckel/dp/8131705579/)
    - [Effective Java](http://www.amazon.in/Effective-Java-Series-Joshua-Bloch/dp/0321356683/)

## IDEs
1. [Eclipse](http://www.eclipse.org/downloads/)
	- Offers a wide variety of plugins and therefore extensible.
2. [Netbeans](https://netbeans.org/)
	- The official(supported by Oracle) IDE.
3. [IntelliJ Idea](https://www.jetbrains.com/idea/download/)
	- Paid IDE, offers a community edition.
